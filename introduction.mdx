## What is Conar?

Conar is an open-source, AI-powered database management tool designed to simplify how you interact with your databases. Whether you're managing PostgreSQL, MySQL, MSSQL, or ClickHouse, Conar provides a modern interface that combines powerful features with an intuitive user experience.

Available as both a desktop application and web app, Conar stores your connections securely in the cloud with encrypted connection strings and password protection, while AI assistance helps you write and optimize SQL queries.

## Key Features

<AccordionGroup>
  <Accordion title="Secure & Open Source" icon="lock">
    - **Open-source codebase** - Full transparency with AGPL-3.0 license
    - **Encrypted connection strings** - Your credentials are encrypted at rest
    - **Password protection** - Optional password storage with secure encryption
    - **Cloud sync without passwords** - Sync connections without storing sensitive credentials
  </Accordion>

  <Accordion title="Multi-Database Support" icon="database">
    Currently supported databases:
    - **PostgreSQL** - Full support for all PostgreSQL features
    - **MySQL** - Complete MySQL compatibility
    - **MSSQL** - Microsoft SQL Server integration
    - **ClickHouse** - High-performance analytics database
    
    Coming soon:
    - SQLite
    - MongoDB
  </Accordion>

  <Accordion title="AI-Powered Features" icon="sparkles">
    - **Intelligent SQL assistance** - Get help writing queries in natural language
    - **Multiple AI models** - Choose from Anthropic, OpenAI, Gemini, and XAI
    - **Smart filtering** - Create filters using conversational prompts
    - **Query optimization** - AI suggestions for better performance
  </Accordion>

  <Accordion title="Modern Tech Stack" icon="code">
    Built with cutting-edge technologies:
    - **Frontend**: React, TypeScript, TailwindCSS, shadcn/ui
    - **Router & State**: TanStack Start/Router/Query/Form/Virtual
    - **Desktop**: Electron, ToDesktop
    - **Backend**: Hono, Drizzle ORM, Better Auth
    - **AI**: AI SDK with multiple providers
    - **Build Tool**: Vite, Bun
  </Accordion>
</AccordionGroup>
