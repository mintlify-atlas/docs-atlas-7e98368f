The `@conar/ui` package is a comprehensive UI library containing React components, custom hooks, and utilities built on top of shadcn/ui and Radix UI primitives.

## Installation

This package is internal to the monorepo:

```json
{
  "dependencies": {
    "@conar/ui": "workspace:*"
  }
}
```

**Props:**

```typescript
interface ButtonProps extends ComponentProps<'button'> {
  variant?: 'default' | 'destructive' | 'outline' | 'secondary' | 'ghost' | 'link'
  size?: 'default' | 'sm' | 'lg' | 'icon'
  asChild?: boolean
}
```

**Implementation:**

```typescript
export function Button({
  className,
  variant,
  size,
  type = 'button',
  asChild = false,
  ...props
}: ButtonProps) {
  const Comp = asChild ? Slot : 'button'

  return (
    <Comp
      type={type}
      className={cn(buttonVariants({ variant, size }), className)}
      {...props}
    />
  )
}
```

## Hooks Collection

### useToggle

Simple boolean toggle hook.

From `packages/ui/src/hookas/use-toggle.ts:3`:

```typescript
import { useToggle } from '@conar/ui/hookas/use-toggle'

function Component() {
  const [isOpen, toggleOpen] = useToggle(false)
  
  return (
    <>
      <Button onClick={() => toggleOpen()}>Toggle</Button>
      <Button onClick={() => toggleOpen(true)}>Open</Button>
      <Button onClick={() => toggleOpen(false)}>Close</Button>
      {isOpen && <div>Content</div>}
    </>
  )
}
```

**Implementation:**

```typescript
export function useToggle(initialValue = false) {
  const [value, setValue] = useState(initialValue)

  const toggle = useCallback((value?: boolean) => {
    setValue(prev => value ?? !prev)
  }, [])

  return [value, toggle] as const
}
```

## Utilities

### cn()

Combine Tailwind classes with conflict resolution.

From `packages/ui/src/lib/utils.ts:5`:

```typescript
import { cn } from '@conar/ui/lib/utils'

// Merge classes
const className = cn(
  'px-4 py-2',
  'bg-blue-500',
  isActive && 'bg-green-500',
  props.className
)

// Result: 'px-4 py-2 bg-green-500' (conflicts resolved)
```

**Implementation:**

```typescript
import type { ClassValue } from 'clsx'
import { clsx } from 'clsx'
import { twMerge } from 'tailwind-merge'

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs)).replace(/\s+/g, ' ').trim()
}
```

## Best Practices

### Component Composition

```typescript
// Good ✓ - Compose from primitives
<Dialog>
  <DialogTrigger asChild>
    <Button variant="outline">Open</Button>
  </DialogTrigger>
  <DialogContent>
    <DialogHeader>
      <DialogTitle>Title</DialogTitle>
    </DialogHeader>
  </DialogContent>
</Dialog>

// Bad ✗ - Don't create monolithic components
<Dialog open onOpenChange title="Title" trigger={<Button />}>
```

### Class Merging

```typescript
// Good ✓ - Use cn() for class merging
function Component({ className }: { className?: string }) {
  return <div className={cn('base-classes', className)} />
}

// Bad ✗ - Don't concatenate strings
function Component({ className }: { className?: string }) {
  return <div className={`base-classes ${className}`} />
}
```

### Hooks

```typescript
// Good ✓ - Use provided hooks
const [isOpen, toggle] = useToggle()
const isMobile = useMediaQuery('(max-width: 768px)')

// Bad ✗ - Don't reinvent the wheel
const [isOpen, setIsOpen] = useState(false)
const toggle = () => setIsOpen(prev => !prev)
```
