## Get Started in 3 Steps

This guide will take you from installation to executing your first database query in just a few minutes.

<Steps>
  <Step title="Install Conar">
    Download and install the desktop app for your platform:

    <Tabs>
      <Tab title="macOS">
        **Option 1: Direct Download**
        
        Download the appropriate installer:
        - [Apple Silicon (M1/M2/M3)](https://download.conar.app/mac/dmg/arm64)
        - [Intel](https://download.conar.app/mac/dmg/x64)

        **Option 2: Homebrew**
        ```bash
        brew install --cask conar
        ```
      </Tab>
      <Tab title="Windows">
        Download the Windows installer:
        - [Windows (.exe)](https://download.conar.app/windows/nsis/x64)

        Run the installer and follow the on-screen instructions.
      </Tab>
      <Tab title="Linux">
        Choose your preferred package format:
        - [Debian (.deb)](https://download.conar.app/linux/deb/x64)
        - [AppImage](https://download.conar.app/linux/appImage/x64)
        - [RPM](https://download.conar.app/linux/rpm/x64)
        
        **Arch Linux (AUR)**:
        ```bash
        yay -S conar-bin
        ```
      </Tab>
      <Tab title="Web App">
        Access Conar directly in your browser:
        
        <Card
          title="Launch Web App"
          icon="globe"
          href="https://conar.app"
        >
          No installation required
        </Card>
      </Tab>
    </Tabs>

    <Note>
      The desktop app provides the best performance and offline capabilities, while the web app offers instant access from any device.
    </Note>
  </Step>

  <Step title="Create Your First Connection">
    Once you've launched Conar, you'll create your first database connection.

    ### Sign Up or Sign In

    First, create an account or sign in to sync your connections across devices.

    <Info>
      Your account is required to enable cloud sync and AI features. All connection strings are encrypted before being stored.
    </Info>

    ### Add a Database Connection

    <Steps>
      <Step title="Select Database Type">
        Click **Create Connection** and choose your database type:
        - PostgreSQL
        - MySQL
        - MSSQL
        - ClickHouse
      </Step>

      <Step title="Enter Connection Details">
        Provide your database connection string. The format depends on your database type:

        <CodeGroup>
          ```text PostgreSQL
          postgresql://username:password@hostname:5432/database
          ```

          ```text MySQL
          mysql://username:password@hostname:3306/database
          ```

          ```text MSSQL
          mssql://username:password@hostname:1433/database
          ```

          ```text ClickHouse
          clickhouse://username:password@hostname:8123/database
          ```
        </CodeGroup>

        <Tip>
          You can use localhost or 127.0.0.1 for databases running on your machine.
        </Tip>
      </Step>

      <Step title="Test the Connection">
        Click **Test Connection** to verify Conar can connect to your database.

        ```typescript
        // Behind the scenes, Conar executes:
        executeSql({
          type: ConnectionType.Postgres,
          connectionString: 'postgresql://...',
          sql: 'SELECT 1'
        })
        ```

        <Check>If successful, you'll see a confirmation message.</Check>
      </Step>

      <Step title="Save Your Connection">
        Configure connection details:

        - **Name**: Give your connection a memorable name (auto-generated by default)
        - **Label**: Add an environment label like "Local", "Dev", "Staging", or "Prod"
        - **Color**: Choose a color for easy identification
        - **Cloud Sync**: Choose whether to sync the password to the cloud

        <Warning>
          If you choose to **not** sync the password, you'll need to enter it each time you use this connection on a new device.
        </Warning>

        Click **Save Connection** to complete the setup.
      </Step>
    </Steps>

    ### Connection Sync Options

    Conar offers two sync modes:

    <CardGroup cols={2}>
      <Card title="Full Sync" icon="cloud-check">
        Syncs connection string including password (encrypted). Best for personal databases.
      </Card>
      <Card title="Sync Without Password" icon="cloud">
        Syncs connection details but requires password entry on each device. Best for production databases.
      </Card>
    </CardGroup>
  </Step>

  <Step title="Execute Your First Query">
    Now that your connection is set up, let's explore your data.

    ### Browse Tables

    1. Click on your connection in the sidebar
    2. Select a schema from the dropdown
    3. Click on any table to view its data

    <img
      src="https://conar.app/demo-light-1.png"
      alt="Table browser interface"
      className="rounded-lg border"
    />

    ### Use the SQL Runner

    Navigate to the **SQL Runner** tab to write and execute custom queries.

    **Write a query manually:**

    ```sql
    SELECT * FROM users
    WHERE created_at > NOW() - INTERVAL '7 days'
    ORDER BY created_at DESC
    LIMIT 10;
    ```

    Click **Run** or press `Cmd/Ctrl + Enter` to execute.

    ### Use AI to Generate Queries

    The real power of Conar comes from AI assistance:

    1. Click the **Chat** icon in the SQL Runner
    2. Type a natural language request:

    <CodeGroup>
      ```text Example 1
      Show me all users who signed up in the last week
      ```

      ```text Example 2
      Find orders with total amount greater than $1000
      ```

      ```text Example 3
      Get the top 10 customers by order count
      ```
    </CodeGroup>

    3. The AI will generate the SQL query for you
    4. Review the query and click **Run** to execute

    <Tip>
      You can choose your preferred AI model in settings. Conar supports Anthropic, OpenAI, Gemini, and XAI.
    </Tip>

    ### Apply Smart Filters

    When viewing table data, use AI-powered filtering:

    1. Click the filter input at the top of the table
    2. Type a natural language filter like:
       - `price between 100 and 500`
       - `created_at after 2024-01-01`
       - `status equals active`
    3. Press Enter and watch the filters apply automatically

    The AI converts your input into SQL filter conditions:

    ```typescript
    // Your input: "price between 100 and 500"
    // Generated filters:
    [
      { column: 'price', operator: '>=', value: '100' },
      { column: 'price', operator: '<=', value: '500' }
    ]
    ```
  </Step>
</Steps>
